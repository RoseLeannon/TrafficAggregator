# Private Traffic Aggregator - Frontend

Modern Next.js frontend for the Private Traffic Aggregator smart contract.

## Tech Stack

- Next.js 16 + TypeScript
- Wagmi + RainbowKit  
- Tailwind CSS + Radix UI
- Vercel Deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
npm run dev
```

Visit http://localhost:3000

## Deploy to Vercel

```bash
vercel
```
